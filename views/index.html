API Visitor - User Guide

Status Server: Active

Information: Jika ada error dengan api ini sila lapor ke Zeyz

Daftar Endpoint

  • GET /api/domain/register: Mendaftarkan Domain baharu.
  • GET /api/domain/delete: Menghapus Domain
  • GET /api/views: Mendapatkan jumlah visitor harian dan total berdasarkan domain.
  • GET /api/views/add: Menambah visitor berdasarkan domain

Menggunakan Endpoint GET /api/domain/register

Anda dapat menggunakan endpoint ini untuk mendaftarkan api visitor baru berdasarkan domain dan key. Gunakan parameter query domain untuk menentukan domain yang ingin Anda daftarkan, dan Gunakan parameter query key untuk menentukan kunci / password yang ingin Anda daftarkan atau kebutuhan untuk delete domain

Contoh penggunaan:

https://visitor.api.zeyzofc.com/api/domain/register?domain=google.com&key=kunci_anda

Respons:

{
    "message": "User registered successfully",
    "user": {
        "total": 0,
        "daily": 0,
        "domain": "google.com",
        "key": "kunci_anda"
    }
}

Catatan: Pastikan untuk selalu menggunakan domain yang unik saat menambahkan atau memeriksa pengunjung.

Menggunakan Endpoint GET /api/domain/delete

Anda dapat menggunakan endpoint ini untuk menghapus data visitor berdasarkan domain dan key. Gunakan parameter query domain untuk menentukan domain yang ingin Anda delete, dan gunakan parameter query key untuk mengesahkan kunci / password yang diperlukan untuk menghapus data (keamanan tambahan).

Contoh penggunaan:

https://visitor.api.zeyzofc.com/api/domain/delete?domain=google.com&key=kunci_anda

Respons:

{
  "message": "Domain deleted successfully."
}

Catatan: Pastikan untuk selalu menggunakan domain yang unik saat menambahkan atau memeriksa pengunjung.

Menggunakan Endpoint GET /api/views

Anda dapat menggunakan endpoint ini untuk mendapatkan jumlah visitor harian / total pengunjung berdasarkan domain (untuk visitor harian akan ter-reset setiap harinya). Gunakan parameter query domain untuk menentukan domain yang ingin Anda cek.

Contoh penggunaan:

https://visitor.api.zeyzofc.com/api/views?domain=google.com

Respons:

{
  "domain": "google.com",
    "total": 153454,
    "daily": 345235236
}

Catatan: Pastikan untuk selalu menggunakan domain yang unik saat menambahkan atau memeriksa pengunjung.

Menggunakan Endpoint GET /api/views/add

Anda dapat menggunakan endpoint ini untuk menambahkan jumlah visitor harian / total pengunjung berdasarkan domain (untuk visitor harian akan ter-reset setiap harinya). Gunakan parameter query domain untuk menentukan domain yang ingin Anda tambahkan visitornya.

Contoh penggunaan:

https://visitor.api.zeyzofc.com/api/views/add?domain=google.com

Respons:

{
    "message": "Visitor added successfully",
    "domain": "google.com",
    "total": 1,
    "daily": 1
}

Catatan: Pastikan untuk selalu menggunakan domain yang unik saat menambahkan atau memeriksa pengunjung
